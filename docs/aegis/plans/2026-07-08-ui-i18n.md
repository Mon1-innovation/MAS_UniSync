# UI i18n Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use aegis:subagent-driven-development (recommended) or aegis:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the web portal Chinese by default with an English language switch, and translate the MAS submod UI to fixed Chinese.

**Architecture:** Add `i18next` and `react-i18next` as the web portal localization layer. Keep translations in focused resource files, wire the provider at app startup, and translate component/page display strings without changing API contracts. The submod keeps internal machine values unchanged and maps only visible text to Chinese.

**Tech Stack:** React 18, Vite, TypeScript, Primer React, `i18next`, `react-i18next`, Ren'Py screen language/Python.

**Baseline / Authority Refs:** `docs/aegis/specs/2026-07-08-ui-i18n-design.md`, `docs/design-plan.md`, `README.md`.

**Compatibility Boundary:** Routes, API payloads, JSON keys, persisted user cache, profile key values, audit action codes, and submod protocol constants must remain unchanged.

**Verification:** `cd frontend; npm test; npm run typecheck` and `pytest tests/test_submod_packaging.py tests/test_submod_client_core.py`.

Create `frontend/src/i18n/index.ts`:

```ts
import i18n from 'i18next'
import {initReactI18next} from 'react-i18next'
import {resources} from './resources'

export const LANGUAGE_STORAGE_KEY = 'mas_unisync_language'
export type SupportedLanguage = keyof typeof resources

export function isSupportedLanguage(value: string | null): value is SupportedLanguage {
  return value === 'zh' || value === 'en'
}

const storedLanguage = isSupportedLanguage(localStorage.getItem(LANGUAGE_STORAGE_KEY))
  ? localStorage.getItem(LANGUAGE_STORAGE_KEY)
  : null

i18n.use(initReactI18next).init({
  resources,
  lng: storedLanguage || 'zh',
  fallbackLng: 'zh',
  interpolation: {escapeValue: false},
})

i18n.on('languageChanged', (language) => {
  if (isSupportedLanguage(language)) {
    localStorage.setItem(LANGUAGE_STORAGE_KEY, language)
  }
})

export {i18n}
```

Import it once in `frontend/src/main.tsx`:

```ts
import './i18n'
```
